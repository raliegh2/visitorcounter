# Changelog

## 1.0.0-rc.1 — 2026-06-23

- Migrated the prototype architecture to Next.js, TypeScript, Supabase Auth, and PostgreSQL.
- Added administrator, usher, and read-only leader roles.
- Added administrator TOTP MFA and AAL2 database checks.
- Added organization isolation and Row-Level Security.
- Added service assignment enforcement.
- Added transaction-safe visitor registration and attendance check-in.
- Added duplicate attendance constraint and auditable corrections.
- Added aggregate reports and logged administrator exports.
- Added configurable contact, visitor, attendance, and audit retention controls.
- Added nonce-based CSP, HSTS, and additional browser security headers.
- Added unit, browser, and database test scaffolding.
- Added deployment, backup, incident, administrator, usher, privacy, and retention documentation.
