# Usher guide

1. Sign in with your own account; never share credentials.
2. Select only the service you are assigned to.
3. Search before registering a visitor.
4. Confirm duplicate names using first-visit and last-seen dates; names are not unique identifiers.
5. For a new visitor, record only name, optional preferred name, and the service date.
6. Record phone/email only when the visitor clearly consents.
7. Confirm the success message after check-in.
8. Ask an administrator to correct errors; ushers cannot alter attendance history.
9. Sign out or lock the tablet when it is unattended.
10. Never enter prayer requests, counseling information, health data, finances, addresses, IDs, or information about minors.
